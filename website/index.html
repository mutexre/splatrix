Features

What you get

One-Click Pipeline

Select a video, adjust settings, click Start. The entire Nerfstudio pipeline runs automatically with real-time progress tracking.

Embedded 3D Viewer

Preview your Gaussian Splat directly inside the app with an embedded WebGL viewer.

Frame Browser

Browse and inspect extracted frames.

Project Persistence

Save and restore projects. Resume interrupted processing from any completed stage. Work on multiple projects simultaneously.

ETA Tracking

Real-time estimated time remaining for each processing stage.

Standard Export

Export to PLY format compatible with all major 3D viewers and editors.

Pipeline

Automated stages

Each stage runs sequentially with detailed progress. Resume from any stage if interrupted.

1

Frame Extraction

Extract frames from video with configurable frame count

2

Feature Extraction

Detect visual features using COLMAP SIFT

3

Feature Matching

Match features across frame pairs

4

Reconstruction

Sparse 3D reconstruction + camera poses

5

Training

Train splatfacto Gaussian Splat model

6

Export

Export trained model to PLY format

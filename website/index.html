Video to Gaussian Splats

Drop a video, click Start, and get a photorealistic 3D scene. Desktop UI with real-time progress tracking and an embedded 3D viewer.

Get Started How it works
Terminal
Install curl -fsSL https://mutexre.github.io/splatrix/install.sh | bash
Run ~/.splatrix/bin/splatrix
Install irm https://mutexre.github.io/splatrix/install.ps1 | iex
Run ~\.splatrix\bin\splatrix.bat
Features

Everything you need

From video input to 3D Gaussian Splat output — no command-line expertise required.

▶

One-Click Pipeline

Select a video, adjust settings, click Start. The entire Nerfstudio pipeline runs automatically with real-time progress tracking.

◼

Embedded 3D Viewer

Preview your Gaussian Splat directly inside the app using a WebGL-based viewer powered by GaussianSplats3D.

📷

Frame Browser

Inspect extracted frames in a responsive grid view with adjustable thumbnail sizes and fit modes.

💾

Project Persistence

Save and restore projects. Resume interrupted processing from any completed stage. Work on multiple projects simultaneously.

⏰

ETA Tracking

Real-time estimated time remaining for each processing stage, so you always know when your splat will be ready.

🔧

Flexible Export

Export to standard PLY format compatible with all major Gaussian Splatting viewers and editors.

Pipeline

Six automated stages

Each stage runs sequentially with detailed progress. Start from any completed stage to iterate quickly.

1

Frame Extraction

Extract frames from video at configurable intervals

2

Feature Extraction

Detect visual features using COLMAP SIFT

3

Feature Matching

Match features across frame pairs

4

Reconstruction

Sparse 3D reconstruction + camera poses

5

Training

Train splatfacto Gaussian Splat model

6

Export

Export trained model to PLY format
